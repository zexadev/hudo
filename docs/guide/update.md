# 自我更新

hudo 支持自我更新，无需重新下载安装脚本。

## 更新 hudo

```powershell
hudo update
```

hudo 会自动检查 GitHub Releases 上的最新版本，若有更新则下载并替换自身。

## 更新流程

1. 查询 GitHub API 获取最新版本号
2. 与当前版本对比
3. 若有新版本，下载新的 `hudo.exe`
4. 将当前 exe 重命名为 `.old`，新 exe 替换到原路径
5. 启动后台进程清理 `.old` 文件

## 已是最新版本

若当前已是最新版本，hudo 会提示：

```
已是最新版本 vX.X.X
```

## 常见问题

**更新后终端关闭了？**

这是正常现象，hudo 在替换自身 exe 时会关闭当前进程，重新打开终端即可。
