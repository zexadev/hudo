# 工具列表

hudo 支持安装以下工具，按分类列出：

## 版本控制 & CLI

| 工具 | 说明 |
|------|------|
| [Git](./git) | 分布式版本控制系统 |
| [GitHub CLI](./gh) | GitHub 官方命令行工具 |

## 语言运行时

| 工具 | 说明 |
|------|------|
| [Node.js](./nodejs) | JavaScript 运行时（通过 fnm 管理版本） |
| [Bun](./bun) | 高性能 JavaScript 运行时 |
| [Python](./python) | Python 环境（通过 uv 管理） |
| [Miniconda](./miniconda) | Python 科学计算环境（Conda 包管理） |
| [Go](./go) | Go 语言 |
| [Rust](./rust) | Rust 语言（通过 rustup 安装） |

## JVM 生态

| 工具 | 说明 |
|------|------|
| [JDK](./jdk) | Java 开发工具包（Eclipse Temurin） |
| [Maven](./maven) | Java 构建工具 |
| [Gradle](./gradle) | Java 构建工具 |

## 数据库

| 工具 | 说明 |
|------|------|
| [MySQL](./mysql) | 关系型数据库 |
| [PostgreSQL](./pgsql) | 关系型数据库 |

## IDE & 编辑器

| 工具 | 说明 |
|------|------|
| [VS Code](./vscode) | 微软 Visual Studio Code |
| [PyCharm](./pycharm) | JetBrains Python IDE |

## 系统工具

| 工具 | 说明 |
|------|------|
| [MinGW](./mingw) | Windows GCC 编译器工具链 |
| [Google Chrome](./chrome) | Google Chrome 浏览器（企业版静默安装） |
| [Claude Code](./claude-code) | Anthropic Claude Code CLI |
