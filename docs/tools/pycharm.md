# PyCharm

JetBrains PyCharm Community Edition，Python 专业 IDE。

## 安装

```powershell
hudo install pycharm
```

安装到 `{install_root}\ide\pycharm\`，自动获取最新版本。

## 安装后

启动 PyCharm：

```powershell
# 直接运行
{install_root}\ide\pycharm\bin\pycharm64.exe
```

## 卸载

```powershell
hudo uninstall pycharm
```

## 配置文件版本

```toml
[versions]
# 不填则自动获取最新版
pycharm = "2024.3.5"
```
